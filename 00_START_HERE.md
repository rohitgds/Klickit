# Start KlickIt in Cursor

## What this package contains

- All ten blueprint files
- One master bootstrap prompt
- 55 small development phases
- Persistent Cursor rules and commands
- Progress, testing, decision and milestone files
- Provider exit, account transfer, clean rebuild and future sale/handover files
- Subscription/API/MCP setup checklists

## Steps

1. Extract this ZIP into the empty folder where you want the KlickIt project.
2. Install and open Cursor.
3. Open the extracted project folder in Cursor.
4. Open `MASTER_CURSOR_BOOTSTRAP_PROMPT.md`.
5. Copy the complete prompt.
6. Open Cursor Agent and paste it.
7. Let Cursor perform read-only inspection first.
8. Approve administrator commands, MCP installations, logins, paid services or destructive actions only after Cursor explains them.
9. Do not paste passwords or API keys into normal chat.
10. At a milestone, complete the click-by-click tests.
11. Type `APPROVE MILESTONE` only when satisfied.

Cursor must stop after the first setup milestone. It must not start feature coding in the first run.


## If development has already started

Use `KLICKIT_PORTABILITY_RETROFIT_PROMPT.md` instead of restarting the project. It audits and improves portability without deleting completed work.
