# Blueprint Manifest

These ten files are the original user-provided requirements. Treat them as read-only.

Priority rules:

1. Document 10 overrides documents 01–09 only where it explicitly records a newer decision.
2. A genuine contradiction, ambiguity, safety concern, or financial uncertainty is a mandatory stop condition.
3. Never silently rewrite these original files.
4. Record clarifications and accepted changes under `blueprints/amendments/` and `docs/DECISION_LOG.md`.
5. KlickIt is the new product. DrKlick is only the authorised reference system for migration discovery.
