# Sale and Handover Checklist

## Product and IP

- [ ] Business owns source code or has written assignment
- [ ] Repository history is complete
- [ ] Dependencies and licenses are documented
- [ ] No copied DrKlick code, UI or private implementation
- [ ] Brand, domain and proprietary assets have ownership records

## Technical transfer

- [ ] Clean rebuild passes
- [ ] Database and file export/import pass
- [ ] Provider inventory is current
- [ ] Secrets can be rotated
- [ ] Account transfer is documented
- [ ] Backups and restore drills are current
- [ ] Known issues and technical debt are disclosed
- [ ] Operating runbooks are complete

## Data transfer

- [ ] Tenant isolation is documented
- [ ] Export formats are documented
- [ ] Retention/deletion policies are documented
- [ ] No real patient data is included in development transfer
