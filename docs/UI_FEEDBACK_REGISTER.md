## Severity guide

| Level | Meaning |
|---|---|
| 1 | Blocks workflow or data safety — fix before next module |
| 2 | Major usability or wrong business behaviour — fix before pilot |
| 3 | Cosmetic or minor wording — can defer with owner approval |
| 4 | Enhancement request — log only |

## How to report (owner)

1. Note the screen reference (example: `UI-PAT-001`).
2. Describe what you clicked and what happened.
3. Describe what you expected.
4. Add a screenshot if possible.
5. Send to the agent; it will be recorded here and in `docs/USER_FEEDBACK.md`.
